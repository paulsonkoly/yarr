# Yarr

This application is the workhorse behind the rubydoc bot on freenode.net #ruby.

## Installation

```bash
$ cd yarr
$ bundle install
```

## Configuration

Write configuration in $HOME/.config/yarr/yarr.yml.

## Usage

To run the application you have to run the following command:

```bash
$ bundle exec yarr
```
